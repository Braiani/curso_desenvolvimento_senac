Atividades de Javascript

  1. Faça um programa que receba uma temperatura e transforme de Fahrenheit para Celsius. A formula de conversão de Fahrenheit para Celsius é a seguinte: C = (5/9) * (F – 32). Utilize o prompt do JS para captar os dados, e utilize o alerta para mostrar. Mostrar 0 casas decimais.
  2. Faça um programa que leia o nome de um produto, a quantidade comprada, o valor unitário e o percentual de desconto a ser aplicado para o pagamento. Imprima na tela com o alert o nome do produto e o valor total da venda.
  3. Faça um programa que leia um valor em reais e calcule o valor equivalente em dólares. O usuário deve informar, além do valor em reais da compra, o valor da cotação do dólar.
